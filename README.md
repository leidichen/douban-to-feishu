# 🎬 Douban to Feishu Sync (豆瓣 → 飞书同步助手)

![License](https://img.shields.io/badge/license-MIT-blue.svg) ![Version](https://img.shields.io/badge/version-1.1.2-green.svg)

一个精致、高效的 Chrome 浏览器扩展，致力于将你的豆瓣书影音数据一键无缝同步到飞书多维表格（Feishu Base）。打造属于你自己的个人文化数据库，从未如此简单。

## ✨ 核心亮点

*   🚀 **一键同步**：在豆瓣详情页点击图标，自动抓取电影/图书元数据（标题、评分、导演/作者、简介、封面等）。
*   🧩 **智能映射**：无需严格对应字段名！插件内置智能别名库，只要你的飞书表格字段叫“片名”、“书名”或“Title”，它都能自动识别并填入。
*   🛠 **一键建表**：懒得手动建表？插件内置“一键初始化”功能，自动在飞书为你创建包含所有必要字段的完美表格。
*   🛡 **安全隐私**：基于飞书官方开放平台接口，数据直连你的飞书空间，安全可靠。
*   💅 **现代化 UI**：简洁美观的界面设计，支持电影与图书双模式切换。

## 📥 安装指南

由于目前插件暂未上架 Chrome 商店，你需要通过“开发者模式”安装：

1.  **下载代码**：点击右上角的 `Code` -> `Download ZIP` 下载并解压本项目。
2.  **打开扩展管理**：在 Chrome 地址栏输入 `chrome://extensions/`。
3.  **开启开发者模式**：打开右上角的开关。
4.  **加载插件**：点击左上角的“加载已解压的扩展程序”，选择本项目中的 `douban-to-feishu` 文件夹。

## ⚙️ 快速配置

### 1. 准备飞书应用 🤖
你需要一个飞书“自建应用”来充当搬运工：
1.  登录 [飞书开放平台](https://open.feishu.cn/app)。
2.  创建企业自建应用，获取 **App ID** 和 **App Secret**。
3.  在“权限管理”中开通 `bitable:app:read` 和 `bitable:app:write` 权限。
4.  发布应用版本。

### 2. 准备多维表格 📊
1.  在飞书新建一个多维表格。
2.  复制表格链接（URL）。

### 3. 插件设置 🔧
点击插件图标 -> **设置**：
*   填入 App ID 和 App Secret。
*   粘贴多维表格链接。
*   点击 **“初始化电影表/图书表”**（强烈推荐！省去手动建字段的麻烦）。

## 📖 使用方法

1.  打开任意豆瓣电影或图书条目页（例如：[《肖申克的救赎》](https://movie.douban.com/subject/1292052/)）。
2.  点击插件图标，选择对应类型（电影/图书）。
3.  点击 **“获取信息”** 预览抓取到的数据。
4.  点击 **“保存到飞书”**，等待绿色成功提示即可！✅

## 🛠 支持字段

插件会自动尝试匹配以下字段（不区分大小写）：

| 字段类型 | 飞书字段名示例 (任选其一) |
| :--- | :--- |
| **标题** | 标题, 书名, 片名, Title, Name |
| **评分** | 评分, 个人评分, Rating, Score |
| **日期** | 上映日期, 出版日期, Date, Pubdate |
| **简介** | 简介, 短评, Comment, Summary |
| **链接** | 链接, URL, Link |
| **封面** | 封面, Cover, Image (文本/附件) |
